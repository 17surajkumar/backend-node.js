REST API - JSON

GET /users - list all the users (HTML)

GET /api/users - list all the users (JSON)
GET /api/users/1 - Get the user with Id 1

Dynamic Path Parameters:
GET /api/users/:id 
:id -> this is the variable

POST /api/users - Create new user

PATCH /api/users/1 - Edit the user with the ID 1

DELETE /api/users/1 - Delte the user with the Id 1
